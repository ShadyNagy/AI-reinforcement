### how to use multi env on Anaconda
Python 2.7 and 3.9
Use: Anaconda Prompt (anaconda3)

```powershell
conda create -n py27 python=2.7 
activate py27

conda create -n py39 python=3.9
activate py39

you may leave the environment back to your global env by typing
deactivate py27 
or 
deactivate py39
```
